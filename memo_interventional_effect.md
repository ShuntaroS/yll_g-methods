疫学における **generalized intervention effect（一般化介入効果）** は，介入の効果を「特定の固定された介入」ではなく，「あるクラスに属する介入全体」に対して定義する枠組みです．近年の因果推論（特に stochastic intervention や dynamic treatment regime の文脈）で重要な概念です．

------

## 1．背景：従来の因果効果との違い

従来の因果効果は，例えば以下のように定義されます：

- **平均因果効果（ATE）**
  [
  E[Y^{a=1}] - E[Y^{a=0}]
  ]

ここでの問題は：

- 介入が「固定値（static）」である（例：全員禁煙 vs 全員喫煙）
- 現実的でない介入設定になりがち

------

## 2．Generalized intervention effect の考え方

generalized intervention effect では，介入を次のように一般化します：

### （1）stochastic intervention

曝露 (A) を確率的に変更する：

[
A \sim g(a \mid L)
]

- (g)：新しい介入下の曝露分布
- (L)：共変量

このときのアウトカム：

[
E[Y^{g}]
]

------

### （2）generalized intervention effect の定義

2つの介入分布 (g_1, g_0) を比較：

[
E[Y^{g_1}] - E[Y^{g_0}]
]

これが generalized intervention effect です．

------

## 3．具体例

### 例：BMIの分布をシフトさせる

- 従来：BMIを「全員25未満にする」
- generalized intervention：
  - BMIを平均で2単位下げる確率的介入

[
g(a \mid l) = f(a - 2 \mid l)
]

このとき：

- 「BMIを少しずつ下げる政策」の効果を評価できる
- 現実的かつ政策的に意味のある推定

------

### 例：薬剤投与確率の変更

- 現状：(P(A=1 \mid L))
- 介入：投与確率を1.5倍にする

[
g(a \mid L) \propto \text{odds}(A=1 \mid L) \times 1.5
]

------

## 4．なぜ重要か

### （1）実現可能性（feasibility）

- 「全員禁煙」のような非現実的介入を避ける

### （2）positivity違反の緩和

- 極端な介入を避けることで support overlap を維持

### （3）政策解釈性

- 「分布を少し変える」＝現実の政策に近い

------

## 5．関連概念との関係

| 概念                            | 関係                   |
| ------------------------------- | ---------------------- |
| static intervention             | 特定値に固定           |
| dynamic regime                  | 共変量に依存する決定則 |
| stochastic intervention         | 分布そのものを変える   |
| generalized intervention effect | 上記を包括する枠組み   |

------

## 6．識別と推定

一般的な識別式（g-computation）：

[
E[Y^g] = E_L \left[ \sum_a E[Y \mid A=a, L] g(a \mid L) \right]
]

推定方法：

- g-computation
- IPW（特に stochastic IPW）
- TMLE（targeted maximum likelihood estimation）

------

## 7．まとめ（直感）

generalized intervention effect とは：

- 「曝露を固定する」のではなく
- 「曝露の分布をどう変えるか」を介入として定義し
- その結果のアウトカム平均を比較するもの

→ より現実的で柔軟な因果効果の定義

------

必要であれば，TMLEやstochastic interventionのR実装例（例：`tmle3`, `ltmle`）も具体的に示せます．

はい，その理解でだいたい合っています．ただし，**g-formulaで定義する介入**としては，「個票レベルで実際に30%を物理的に選んで書き換える」というより，

> **もともと曝露だった人のうち，各層で30%が非曝露になるような介入下での反実仮想平均アウトカム**
> を定義して，それを g-formula で積分計算する

と考えるのがより正確です．

------

## 1．今回の介入の定義

曝露を (A \in {0,1})，共変量を (L)，アウトカムを (Y) とします．

実際の集団との比較対象として，

- もともと (A=0) の人はそのまま (A^*=0)
- もともと (A=1) の人は，30%の確率で (A^*=0)，70%の確率で (A^*=1)

という介入を考えるわけです．

これは stochastic intervention の一種です．

このとき介入後曝露 (A^*) の条件付き分布は

[
P(A^*=1 \mid A=0,L)=0,\qquad
P(A^*=1 \mid A=1,L)=0.7
]

です．
あるいは簡単に，

[
A^* =
\begin{cases}
0 & \text{if } A=0\
\text{Bernoulli}(0.7) & \text{if } A=1
\end{cases}
]

です．

------

## 2．「曝露群からランダムに30%を非曝露にする」でよいか

**概念的には yes** です．
ただし，注意点が2つあります．

### 注意1：単純な再ラベリングだけでは不十分

ただ「曝露群の30%のラベルを0に置き換えて，そのまま観察アウトカムを使う」ではだめです．

なぜなら，介入後にその人のアウトカムは

- 「もともと曝露だったときの観察アウトカム」

ではなく，

- 「もし非曝露になっていたらどうなったか」という反実仮想アウトカム

だからです．

つまり，30%選んで (A=0) に変えた人については，観察された (Y) をそのまま使うのでなく，
**(E[Y \mid A=0,L])** に基づいて介入後アウトカムを評価しなければいけません．
ここが g-formula の役割です．

------

### 注意2：ランダム化は「介入の表現」であって，推定の本体ではない

「曝露群からランダムに30%を選ぶ」というのは，介入を直感的に表したものです．
g-formula では通常，実際に何度もランダムサンプリングして平均を取らなくても，**期待値を直接計算**できます．

------

## 3．g-formula での平均アウトカムの書き方

まずアウトカムモデルを

[
m(a,l)=E[Y\mid A=a,L=l]
]

とします．

実際の集団の平均アウトカムは

[
E[Y]=E{m(A,L)}
]

です．

一方，介入後集団の平均アウトカムは

[
E[Y^{g}] = E\left[0.3\cdot m(0,L) + 0.7\cdot m(1,L)\mid A=1\right]P(A=1)

- E\left[m(0,L)\mid A=0\right]P(A=0)
  ]

ですが，もっと見やすく書くと

# [ E[Y^{g}]

E\Big[
I(A=0),m(0,L)
+
I(A=1){0.3,m(0,L)+0.7,m(1,L)}
\Big]
]

です．

これはさらに

# [ E[Y^{g}]

E\Big[
(1-0.3A)m(0,L)+0.7A,m(1,L)
\Big]
]

とも書けます．

つまり，**曝露だった人では (m(0,L)) と (m(1,L)) の30:70の混合**になり，
**非曝露だった人では (m(0,L)) のみ**になります．

------

## 4．実装上の直感

実装は次の流れです．

### ステップ1

アウトカム回帰モデルを当てる．

[
E[Y\mid A,L]
]

を推定する．

### ステップ2

各個体について2つの予測値を作る．

- ( \hat m(0,L_i) )
- ( \hat m(1,L_i) )

### ステップ3

介入後アウトカム期待値を各個体で作る．

- もし実際に (A_i=0) なら
  [
  \hat Y_i^{g} = \hat m(0,L_i)
  ]
- もし実際に (A_i=1) なら
  [
  \hat Y_i^{g} = 0.3\hat m(0,L_i)+0.7\hat m(1,L_i)
  ]

### ステップ4

それを全体平均する．

[
\widehat{E[Y^g]}=\frac{1}{n}\sum_{i=1}^n \hat Y_i^g
]

そして，

- リスク差なら
  [
  \widehat{E[Y^g]}-\widehat{E[Y]}
  ]
- リスク比なら
  [
  \widehat{E[Y^g]}/\widehat{E[Y]}
  ]

を計算します．

------

## 5．「実際の集団」との比較とは何か

ここも少しだけ厳密にすると，2通りあります．

### 比較A：観察平均との比較

[
E[Y^g] - E[Y]
]

これは「介入後平均」と「実際に観察された平均」との差です．
実務上よく使えますが，右辺の (E[Y]) は観察データの平均なので，厳密には介入なし反実仮想平均 (E[Y^{\text{natural}}]) と同一視しています．

### 比較B：自然曝露分布下の g-formula 平均との比較

[
E[Y^g]-E[Y^{g_0}]
]

ここで (g_0) は実際の曝露メカニズムです．
理論的にはこちらの方がきれいです．ただし，単一点曝露なら観察平均と実質同じ扱いになることが多いです．

------

## 6．Rでの単純な実装例

連続アウトカムを例にすると，基本形はこうです．

```r
# Y: outcome
# A: exposure (0/1)
# L1, L2: confounders

fit <- glm(Y ~ A + L1 + L2, data = dat, family = gaussian())

# A=0 の下での予測
dat0 <- dat
dat0$A <- 0
m0 <- predict(fit, newdata = dat0, type = "response")

# A=1 の下での予測
dat1 <- dat
dat1$A <- 1
m1 <- predict(fit, newdata = dat1, type = "response")

# stochastic intervention:
# もともとA=1の人の30%をA*=0にする
yg <- ifelse(dat$A == 1,
             0.3 * m0 + 0.7 * m1,
             m0)

# 介入後平均
mean_yg <- mean(yg)

# 観察平均
mean_y_obs <- mean(dat$Y)

# リスク差/平均差
effect_rd <- mean_yg - mean_y_obs

mean_yg
mean_y_obs
effect_rd
```

二値アウトカムでも同じで，`family = binomial()` にして予測確率を使えばよいです．

------

## 7．重要な前提

この解釈が妥当であるには，少なくとも以下が必要です．

- **consistency**
- **exchangeability**
  [
  Y^a \perp A \mid L
  ]
- **positivity**
- アウトカムモデルの適切な指定

また，今回の介入は「曝露者の中から無作為に30%を非曝露化する」ので，
**介入割付が (L) に依存しない stochastic intervention** です．
もし「高齢者では50%，若年者では10%を非曝露化」のようにしたければ，介入確率を (L) 依存にすればよいです．

------

## 8．結論

ご質問への端的な答えは，

- **はい，概念的には「曝露群からランダムに30%を非曝露にして，非曝露群と合わせた集団」と考えてよいです．**
- ただし g-formula では，**その30%の人のアウトカムを観察値のまま使うのではなく，非曝露だった場合の予測アウトカム (E[Y\mid A=0,L]) で置き換えて平均を計算する**，というのが本質です．
