# Estimand-specific wrappers

Thin wrappers around
[`estimate_yll_gformula()`](https://shuntaros.github.io/yll_g-methods/reference/estimate_yll_gformula.md)
that pre-fill the `estimand` argument.

## Usage

``` r
estimate_yll_gformula_ate(...)

estimate_yll_gformula_att(...)

estimate_yll_gformula_atc(...)
```

## Arguments

- ...:

  Arguments forwarded to
  [`estimate_yll_gformula()`](https://shuntaros.github.io/yll_g-methods/reference/estimate_yll_gformula.md).

## Value

Same list structure as
[`estimate_yll_gformula()`](https://shuntaros.github.io/yll_g-methods/reference/estimate_yll_gformula.md).
